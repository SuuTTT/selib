2 · Method genealogy

Who builds on whom (curated lineage; edge = extends / is the deep version of).

Li & Pan 2016 — k-D structural entropy (the source)
  • deDoc 2018 — greedy 2D-SE minimization (Hi-C)
    • SuperTAD 2021 — DP encoding tree (Hi-C TADs)
      • SuperTAD2 2024
      • deTOKI 2021 — TAD-insulation
    • deDoc2 2023 — DP whole-map 2D-SE (Hi-C)
    • pySEAT 2022 — single-cell hierarchy
  • HCSE 2021 — hierarchical clustering via SE
    • BBM — balanced binary merge (encoding-tree baseline)
      • se_hier (this lib) — exact-guarded tree refinement
    • HCSE-graphs 2025 (UAI) — graph version
  • CoDeSEG 2025 — SE-game community detection
    • se_louvain (this lib) — multilevel 2D-SE
  • DeSE — attributed deep SE clustering
  • LSENet — Lorentz/hyperbolic SE clustering
    • se_gnn (this lib) — soft 2D-SE + GCN + Sinkhorn
  • SEP 2022 (ICML) — SE-guided graph pooling
    • SE-GSL — SE graph structure learning
    • SEGA — SE-anchored graph contrastive learning
    • HiTIN — hierarchical text classification
  • SI2E — SE intrinsic exploration (RL)
    • SIDM — SE hierarchical decision making
    • SISA — SE state abstraction
  • Extensions of the definition
    • Flow-based directed SE 2019
    • Multi-Relational SE 2024 (UAI)
    • Incremental SE 2024 (dynamic graphs)
    • Hypergraph SE 2025
    • EDEN 2025 — directed-graph deep SE
  • Social / event / bot apps
    • Hierarchical+Incremental SE events 2024
    • UnDBot — social bot detection
    • SEIN 2026 — open-world events
  • LLM / uncertainty (emerging)
    • SeSE — structural uncertainty / hallucination
    • RAG-SEDE — retrieval structuring
